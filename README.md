# Pokémon App

## Overview
React based Pokémon App to learn about Generation 1 Pokémon!

&nbsp;
## How to run App
First run these commands
```
git clone https://github.com/r-o-e-574/React-Pokemon-App.git
```
```
npm install
```
```
npm run
```
Then follow the URL to see the App
```
http://localhost:3000/
```
